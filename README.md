# Voice English Dictionary (Echo) ✨

Интерактивное приложение для изучения английских слов с помощью голосового ассистента «Салют». Приложение позволяет создавать персональные словари, добавлять слова голосом с автоматическим переводом и проходить тестирование.

## Основные возможности

- Управление словарями: Создание и удаление групп слов (словарей).


- Умное добавление: Вы произносите слово на русском - ассистент сам переводит его на английский (через MyMemory API) и сохраняет в активный словарь.


- Режим тестирования: Голосовая проверка знаний. Ассистент называет русское слово, а вы должны произнести перевод.


- Мультиплатформенность: Работает как обычное React-приложение и как смарт-апп в экосистеме Сбера.

## Технологии

- Frontend: React.js
- Assistant SDK: @salutejs/client 
- Logic: SmartApp Code (сценарии на языке DSL).
- API: MyMemory Translated API для автоматического перевода.
